# 지수 법칙 Ⅱ [나눗셈, 분수, 괄호]

### 정리

<aside>
💡 a ≠ 0이고,  m, n이 자연수일 때,

a^m / a^n
 = a^(m - n) [단, m > n]
 = 1 [단, m = n]
 = 1 / a^(n - m) [단, m < n]

</aside>

### 예시

1. a^6 / a^2

    = a^(6 - 2)

    = a^4

2. b^5 / b^3 / b^2

    = b^(5 - 3) / b^2

    = b^2 / b^2

    = 1

3. c^3 / c^7

    = 1 / c ^(7 - 3)

    = 1 / c^4


### 정리 [괄호가 있는 경우]

<aside>
💡 b ≠ 0 이고, m이 자연수일 때,

(ab)^m = a^mb^m
(a/b)^m = a^m/b^m [단, b ≠ 0]

</aside>

### 예시

1. (a^3b^2)^2

    = (a^3)^2(b^2)^2

    = a^6 * b^4

    = a^6b^4

2. (-a)^4 * (-b)^3

    = (-1)^4a^4 * (-1)^3b^3

    = a^4 * -b^3

    = -a^4b^3

3. (ab)^4 * (a/b)^4

    = a^4 * b^4 * a^4 / b^4

    = a^4 * a^4

    = a^(4 + 4)

    = a^8
