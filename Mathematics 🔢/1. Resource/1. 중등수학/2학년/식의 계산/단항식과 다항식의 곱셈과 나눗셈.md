# 단항식과 다항식의 곱셈과 나눗셈

### 정리 [단항식의 곱셈]

<aside>
💡 (a + b) * c = ac + bc

단항식 * 다항식 곱셈은 분배 법칙을 활용해 계산한다.

</aside>

### 예시

1. (2a^2 + 3ab) * a

    = 2a^3 + 3a^2b

2. 2ab(3a^3b + 2ab^2)

    = 6a^3b^2 + 4a^2b^3

3. 4a(2a + 3b) - 2b(a + 3b)

    = 8a^2 + 12ab - 2ab - 6b^2

    = 8a^2 + 10ab - 6b^2


### 정리 [단항식의 나눗셈]

<aside>
💡 (a + b) / c = (a + b) * 1 / c

단항식 * 다항식 나눗셈은 단항식을 역수로 치환하고 곱셈을 한다.

</aside>

### 예시

1. (15ab + 5ab^2) / 5b

    = (15ab + 5ab^2) * 1 / 5b

    = 3a + ab

2. (4a^2b - 6ab^2 + 3ab) / 2ab

    = (4a^2b - 6ab^2 + 3ab) * 1 / 2ab

    = 2a - 3b + 3/2

3. (4a^2b + 3ab^2) / 1 / 2ab

    = (4a^2b + 3ab^2) * 2ab

    = 8a^3b^2 + 6a^2b^3
