# 다항식의 덧셈과 뺄셈

### 정리

<aside>
💡 다항식의 덧셈과 뺄셈은 문자와 차수가 같은 동류항끼리 계산, 괄호가 있으면 분배법칙을 이용한다.
소, 중, 대괄호 순으로 괄호안의 식을 계산해야한다.

</aside>

### 예시 Ⅰ

1. 3(a + b) - 2(a - b)

    = 3a + 3b - 2a + 2b

    = a + 5b

2. 3a + 2[b + 3{a + 3b - (2b -b)} + 3a]

    = 3a + 2[b + 3{a + 3b - b} + 3a]

    = 3a + 2[b + 3{a + 2b} + 3a]

    = 3a + 2[b + 3a + 6b + 3a]

    = 3a + 2[7b + 6a]

    = 3a + 14b + 12a

    = 15a + 14b


### 예시 Ⅱ

1. (2 - a - 3a^2) + (4a^2 + 2a - 2)

    = 2 - a - 3a^2 + 4a^2 + 2a - 2

    = a^2 + a

2. 3(a^2 + 3a + 3) + 4(a^2 - 3a) -2

    = 3a^2 + 9a + 9 + 4a^2 - 12a -2

    = 3a^2 + 4a^2 - 3a + 7

    = 7a^2 - 3a + 7
