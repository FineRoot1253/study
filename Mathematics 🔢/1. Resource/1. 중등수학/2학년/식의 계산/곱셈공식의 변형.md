# 곱셈공식의 변형

### 정리 Ⅰ [제곱의 합]

<aside>
💡 (x + y)^2 = x^2 + 2xy + y^2
(x - y)^2 = x^2 - 2xy + y^2

x^2 + y^2 = (x + y)^2 - 2xy
x^2 + y^2 = (x - y)^2 + 2xy

</aside>

### 예시

1. 어떤 두 수 x, y의 합이 5이고 곱이 10일 때, x^2 + y^2는?

    x + y =5

    xy = 10,

    x^2 + y^2

    = (x + y)^2 - 2xy

    = 5^2 - 2 * 10

    = 25 - 10

    = 15


### 정리 Ⅱ [합의 제곱, 차의 제곱]

<aside>
💡 (x + y)^2 = x^2 + 2xy + y^2
x^2 + y^2 = (x + y)^2 - 2xy

(x - y)^2 = x^2 - 2xy + y^2
x^2 + y^2 = (x - y)^2 + 2xy

∴ (x + y)^2 - 2xy = (x - y)^2 + 2xy

(x + y)^2 = (x - y)^2 + 4xy
(x - y)^2 = (x + y)^2 - 4xy

</aside>
